# Utilities

This repository contains a static site-style project with the following structure:

- `assets/` — styles, scripts, and images
- `blog/` — blog content (`posts/`)
- `pages/` — standalone pages
- `partials/` — reusable fragments/partials

## Getting Started

You can work with this locally as a standard Git repository.

### Initialize and view status
```
git status
```

### Serve locally (optional)
If you have Node.js installed, you can use a simple static server:
```
npx serve .
```
Or with Python 3:
```
python -m http.server 8080
```
